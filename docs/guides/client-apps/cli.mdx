## Overview

The Formal CLI is part of the [Desktop App](/docs/guides/client-tools/desktop-app) and provides command-line access to Formal resources, authentication, and connection management.

## Installation

The CLI is automatically installed with the Desktop App:

1. Install the [Desktop App](/docs/guides/client-tools/desktop-app)
2. Add to PATH (macOS: Preferences â†’ Install in PATH)
3. Verify: `formal --version`

## Quick Reference

### Authentication

```bash
formal auth login              # Log in to Formal
formal auth credentials        # Get your Formal credentials
formal status                  # Check login status
```

### Resources

```bash
formal ls                      # List all resources (interactive TUI)
formal connect <resource>      # Connect to a resource
formal disconnect <resource>   # Disconnect from a resource
```

### SSH

```bash
formal ssh --sidecar-name <connector>   # SSH to a server
```

### Policy Testing

```bash
formal policy evaluate \
  --inputs '{"user_id":"abc123"}' \
  -r session
```

### Utilities

```bash
formal decrypt --ciphertext <encrypted>  # Decrypt Formal-encrypted data
formal s3 <sidecar> <command>            # S3 operations through Formal
formal help                              # Show help
formal <command> help                    # Command-specific help
```
